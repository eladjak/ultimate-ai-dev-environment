# Authentication (Better Auth)

**Note:** Only use Better Auth for non-Supabase projects. Supabase has built-in auth.

## Skills

| Skill | When to Use |
|-------|-------------|
| `better-auth-best-practices` | Integrating Better Auth, implementation patterns |
| `create-auth-skill` | Creating auth layers with Better Auth |

## Commands (from better-auth plugin)

| Command | When to Use |
|---------|-------------|
| `/providers` | List available auth providers (OAuth, social, etc.) |
| `/explain-error` | Explain Better Auth error codes with solutions |

## Better Auth MCP

| Tool | Purpose |
|------|---------|
| `chat` | Ask questions about Better Auth |
| `search` | Search Better Auth docs and examples |
| `list_files` | List available documentation files |
| `get_file` | Get specific documentation file |

## Features

- Email/password authentication
- OAuth providers (Google, GitHub, Discord, etc.)
- Magic link / passwordless
- Two-factor authentication (2FA)
- Session management
- Role-based access control
