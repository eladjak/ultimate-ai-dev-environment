# Daily Summary

Generate comprehensive daily productivity reports.

## Commands

| Command | Description |
|---------|-------------|
| `/daily-summary` | Generate today's summary |
| `/daily-summary week` | Week overview |
| `/daily-summary save` | Save report to file |

## Report Includes

- Session Tracker accomplishments
- SAGE approvals count
- Skills created/modified
- Git commits made
- Total productivity score

## Integration

Pulls data from:
- Session Tracker
- SAGE state
- Git history
- Development log
