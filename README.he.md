## 🚀 התחלה מהירה

### דרישות מוקדמות

- [Claude Code](https://claude.com/code) מותקן
- Node.js 18+ ו-npm (או bun)
- Git

### התקנה

```bash
# שכפול המאגר
git clone https://github.com/eladjak/ultimate-ai-dev-environment.git
cd ultimate-ai-dev-environment

# התקנת skill-registry-mcp (לב המערכת!)
cd skill-registry-mcp
bun install  # או npm install
bun run build
cd ..

# העתקת סקילים לתיקיית .claude שלך
cp -r skills/* ~/.claude/skills/
cp -r commands/* ~/.claude/commands/
cp -r rules/* ~/.claude/rules/

# הגדרת שרתי MCP
cp config-templates/.mcp.json.template ~/.mcp.json
# ערוך/י את ~/.mcp.json עם הנתיבים האמיתיים שלך
```

### צעדים ראשונים

לאחר ההתקנה, הפעל/י מחדש את Claude Code ונסה/י:

```
"List all available skills"
"Search for skills about React"
"What skill should I use for authentication?"
```

## 💡 תרחישי שימוש

### הבנת קוד סמנטית (מהירה פי 900!)

**לפני (grep):** 45 שניות, הרבה תוצאות חיוביות-שווא
```bash
grep -r "getUserData" .
# מחזיר 347 התאמות כולל הערות, strings...
```

**אחרי (LSP):** 50 אלפיות שנייה, אפס תוצאות חיוביות-שווא
```
"Go to definition of getUserData"
→ ניווט מיידי לפונקציה האמיתית
→ אפס תוצאות חיוביות-שווא מהערות או strings
```

### למידה מתמשכת

```
משתמש: /learn
Claude: מנתח את היסטוריית הסשן...
       מזהה 3 דפוסים חדשים:
       1. שימוש ב-Result types לטיפול בשגיאות
       2. Parallel tool calls לחיפושים
       3. LSP לפני Read
       נשמרו כ-instincts עם רמת ביטחון
```

### פיתוח Multi-Agent

```
משתמש: "צור/צרי אפליקציית todo עם אימות"

אג'נטים משתפים פעולה:
├─ architect: מעצב ארכיטקטורת מערכת
├─ code-reviewer: בודק איכות ואבטחה
├─ security-reviewer: מזהה פגיעויות
└─ tdd-guide: מנחה כתיבת בדיקות

תוצאה: אפליקציה שלמה עם בדיקות ואבטחה!
```
